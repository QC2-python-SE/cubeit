Installation
=============

Requirements
------------

- Python >= 3.8
- NumPy >= 1.20.0
- matplotlib >= 3.10.0
- voila >= 0.5.11
- voila-material >= 0.4.3
- ipywidgets >= 8.2.1
- ipython >= 8.30.0
- ipykernel >= 6.28.0


Installing CubeIt
-----------------

Install from requirements file:

.. code-block:: bash

   pip install -r requirements.txt

Or install as a package:

.. code-block:: bash

   pip install -e .
